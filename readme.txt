Virtual Machine – UNESP/LARS

Este repositório contém a documentação e os scripts associados à máquina virtual
utilizada em atividades acadêmicas e experimentos.

Download VirtualBox
https://www.virtualbox.org/wiki/Downloads
- VirtualBox recomendado: 
	-> 6.1.x compatibilidade com sistemas mais antigos ou addons que ainda não foram atualizados
	-> 7.0.x equilíbrio entre novos recursos e estabilidade
- Instalar Extension Pack (ver menu do VirtualBox)


Download da Máquina Virtual
A imagem da máquina virtual (formato.ova) está disponível em:
https://drive.google.com/drive/folders/1_YtbZtrQYw44xTJuov51cf0XXXtdZDhQ?usp=share_link -- *somente através da conta da UNESP*

Tamanho aproximado: 5 GB  
Formato: OVA (compatível com VirtualBox)

Especificação VM(v1)
- Sistema Operacional: kali-rolling
- Usuário padrão: root
- Senha padrão: unesp
- Memória utilizada - 1024

Especificação VM(v2)
- Sistema Operacional: Debian 10
- Usuário padrão: root
- Senha padrão: unesp
- Memória utilizada - 1024


Como importar a VM
1. Abra o VirtualBox
2. Vá em "Arquivo" → "Importar Appliance"
3. Selecione o arquivo ".ova"
4. Clique em "Importar"
